Introdução ao HTML

Conhecendo as tags básicas

Títulos

Os títulos são usados para definir a estrutura de um documento HTML. Existem seis níveis de títulos, representados pelas tags h1 a h6. A tag h1 é o título mais importante e é usado para o título principal da página.

Subtítulos

Os subtítulos são usados para definir a estrutura de um documento HTML em um nível mais detalhado. A tag h2 é usada para subtítulos, seguida de h3, h4, h5 e h6.

Tags Descontinuadas no HTML5

No HTML5, algumas tags foram descontinuadas por serem consideradas obsoletas ou por terem sido substituídas por outras tags mais eficientes.

  • font: Substituída pela tag span com estilos CSS.
  • center: Substituída pela tag div com estilos CSS.
  • strike: Substituída pela tag del.
  • u: Substituída pela tag span com estilos CSS.
  • big: Substituída pela tag span com estilos CSS.
  • small: Substituída pela tag span com estilos CSS.

É importante notar que, embora essas tags sejam descontinuadas, elas ainda podem ser usadas em alguns casos, especialmente quando se trabalha com conteúdo legado ou com navegadores mais antigos.
